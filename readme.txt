إضافة ووردبريس لإدارة الأبراج السكنية والشقق داخل الموقع العقاري باستخدام نمط Wizard تفاعلي.

== Description ==

تتيح هذه الإضافة إدارة العقارات بشكل سهل وفعال، حيث يمكنك إضافة الأبراج السكنية والشقق وعرضها داخل موقعك.

== Installation ==

1. قم برفع ملفات الإضافة إلى مجلد `/wp-content/plugins/real-estate-manager`، أو قم بتثبيت الإضافة مباشرة من خلال لوحة تحكم ووردبريس.
2. قم بتفعيل الإضافة من خلال صفحة "الإضافات" في ووردبريس.
3. يمكنك الآن إدارة الأبراج والشقق من خلال القائمة الجانبية في لوحة التحكم.
